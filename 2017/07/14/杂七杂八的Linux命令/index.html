杂七杂八的Linux命令

根据进程号，找对应的进程、杀死进程、清理缓存、暂停进程等

根据进程号，找对应的进程

ls -l /proc/PID/exe

杀死进程

Kill:中断一个处理进程(process)
当中断一个前台进程是通常用ctrl+c；对于后台进程用kill命令
kill命令是通过向进程发送指定的信号来结束的。默认为TERM信号。TERM信号将终止所有不能捕获该信号的进程，对于能捕获该信号的进程需要使用kill –9信号，该信号是不能被捕获的。
kill杀终端，只能可以把终端的shell杀死，而退出终端，但终端不关闭
pkill 命令名 可以直接杀死进程
pkill qmail //直接杀死qmail程序进程

查看进程

ps命令进程查看命令
ps命令：process status
-e 显示所有进程
-f 全格式
-h 不显示标题
-l 长格式
-w 宽输出
a 显示终端上的所有进程，包括其他用户的进程
r 只显示正在运行的进程
x 显示没有控制终端的进程

清理Linux系统缓存

echo 1 > /proc/sys/vm/drop_caches

0 – 不释放1 – 释放页缓存2 – 释放文件节点缓存和目录项缓存
3 – 释放所有缓存

进程暂停与恢复

kill -STOP PID
kill -CONT PID
