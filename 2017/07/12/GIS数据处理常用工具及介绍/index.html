shp数据切图

工具存放/app/tiler_program

需要配置的地方

  • 数据准备
    首先将需要切片的web mercator格式的shapefile数据放到data目录下。
    数据最好在QGIS或ArcGIS里建好索引，否则生成会比较慢。

  • 切片程序参数配置

    • 打开 style.json 配置切片的样式。需要注意的几个地方：
      data_name 这个属性要指定为拷贝到 data 目录下的shapefile 的名字。
      res_max 和 res_min 指定的是显示的分辨率的范围，不同级别的分辨率可以在 lod.json 中查找。

    • lod.json中的分辨率等级为从上到下第0级开始

    • 打开 tiler_parameter.json 配置切片的级别（levels）和范围（boundary）

  • 打包程序参数配置
    打开 gen_bundle.bash 修改第18行配置打包的瓦片级别。

使用方法

在切图工具目录下执行 sh gen_bundle.bash或./gen_bundle.bash
想要后台执行脚本可使用 nohup sh gen_bundle.bash或nohup ./gen_bundle.bash

提示：*切图等级从13级开始工作量非常大，可以先切至12级*
