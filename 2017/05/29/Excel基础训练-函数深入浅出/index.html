Excel基础训练-函数深入浅出

公式和函数基本概念及其关系，以及函数的使用方式

公式的基本概念

公式是由用户自行设计对工作表进行计算和处理的计算式。在Excel中以等号开头的式子。

比如，在单元格内键入 =2+2 在按下回车以后便会输出公式的结果
只要拥有运算符号并且符合运算标准，都可以得到相应的运算结果

函数与公式的关系

函数与公式既有区别又互相联系。函数是公式的一部分，函数的出现可以辅助我们进行公式的运算。
比如简单的求和，利用公式需要点击所有想要求和的单元格。而利用SUM函数可以快速的求一部分单元格内数据的总和。

函数的基础概念和录入方式

Excel函数即是预先定义，执行计算、分析等处理数据任务的特殊公式。从字面意思上理解，函数（function），代表了一个功能模块。
以常用的求和函数SUM为例，它的语法是“SUM（number1，number2，．．．．．．）”。其中“SUM”称为函数名称，一个函数只有唯一的一个名称，它决定了函数的功能和用途。函数名称后紧跟左括号，接着是用逗号分隔的称为参数的内容，最后用一个右括号表示函数结束。

函数的录入方式是，首先输入 “=”，然后输入函数名，内容高亮的同时按下Tab键便可插入函数。

基础条件计数

countif函数


想要统计一列数据中某个名称出现的个数

  1. 在想要输出结果的单元格内输入“=countif“按下Tab键

  2. 然后点击上方区域的插入函数

  3. 在出现的对话框第一个选择行选择需要计数的列表，第二个选择行选择需要计数的对象

  4. 在计数后的数据表格右下角双击，可以直接对接下来需要计数的数据进行快速填充

数据核对


想要筛选出未签到的人员名单进行信息核对。

  1. 新建一个辅助列
  2. 在想要输出结果的单元格内输入“=countif“按下Tab键
  3. 然后点击上方区域的插入函数
  4. 在出现的对话框第一个选择行选择需要计数的列表，第二个选择行选择需要计数的对象
  5. 在计数后的数据表格右下角双击，可以直接对接下来需要计数的数据进行快速填充
