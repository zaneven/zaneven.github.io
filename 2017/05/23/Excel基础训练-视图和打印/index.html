Excel基础训练-视图和打印

介绍Excel中快速翻阅工作表、跳转单元格、影子工作簿、冻结窗格、打印设置等技巧

快速翻阅工作表的方式


当工作簿里有特别多的工作表的时候，快速到达指定的工作表的方法


在此处单击鼠标右键->选择工作表即可

快速跳转单元格

  • 双击单元格边框
    双击上下边框可以跳转到单元格所在列的最上方和最下方单元格

  • Ctrl+↑，Ctrl+↓，Ctrl+←，Ctrl+→
    可以达到和双击单元格同样的效果

  • Ctrl+Home/End
    快速跳转到第一个单元格和最后一个单元格

影子工作簿快速核对数据


当想检核对照两个工作簿的数据时，来回点来点去很麻烦，这时可以使用影子工作簿


  • 首先找到视图选项卡->点击新建窗口



  • 生成的新窗口和原来的表格一样，相当于镜像，而且当改动一个表格的内容后，另一个表格的内容同样会发生变化

冻结窗格

在浏览数据的时候，想要固定第一行或者第一列作为属性类型参考


  • 首先找到视图选项卡->点击冻结窗格


  • 冻结窗格有以下三个类型

    • 冻结拆分窗格

    选定单元格，点击冻结拆分窗格后，单元格以上以及左侧的所有行列都被固定

    • 冻结首行
      只固定表格第一行
    • 冻结首列
      只固定表格第一列
