MacOS XAMPP中Apache 启动后闪停

问题描述：

mac启动后,XAMPP下的Apache无法启动,如下：

点击 start 后log里显示

但是很快就跳停，并且没有任何错误提示。

解决办法：

输入：

sudo /Applications/XAMPP/xamppfiles/xampp start

提示：

XAMPP: Starting Apache...fail. XAMPP: Another web server is alreadyrunning.

刚开始以为是端口占用问题，后来发现是提示有另一个web server正在启动，使用sudo apachectl stop提示

/System/Library/LaunchDaemons/org.apache.httpd.plist: Could not find specified service

重启之后仍然无法解决。

发现是自己装了Apple server中占用了localhost，停止了server中的网站功能之后成功启动。
